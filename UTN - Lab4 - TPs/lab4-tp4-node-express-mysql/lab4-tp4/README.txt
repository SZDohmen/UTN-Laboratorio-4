Para levantar la api:
npm start

En caso de modificar el codigo --> actualizar/generar js necesarios:
tsc --watch

[!] Ambos comandos no pueden ejecutarse a la vez

Dependecias utilizadas:
 - ejs
 - express
 - promise-mysql
 - nodemon
 - path
 - typescript

La base de datos debe crearse antes de levantar el proyecto
